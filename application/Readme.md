## Downloading as Ready to Use application 

#### Run this to download all the dependencies
> pip install tkinter docx npttf2utf

#### Run the following script
> pyinstaller --onefile --windowed --icon=icon.ico --add-data "map.json;." main.py        

#### After running this there will be build files, and dist file
#### Make sure there is output folder, because without it the program won't run and will give an error


### Alternatively you can just download the main.exe file 

## Current version supports only Windows (no support for mac/ or other linux distros)
